# Sprints

Use this folder for sprint planning, sprint goals, scope, review notes, and retrospective links.

Create a sprint note from [[../templates/sprint|Sprint Template]].

## Naming

```text
YYYY-MM-DD-sprint-n.md
```

## Sprint Note Should Include

- Sprint goal.
- Selected backlog tasks.
- Capacity or constraints.
- Review summary.
- Retro links or action items.
