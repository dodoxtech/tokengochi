# Decisions

Use this folder for Architecture Decision Records (ADRs) and other durable technical/product decisions.

Create a new decision by copying `docs/templates/decision.md` and naming it:

```text
YYYY-MM-DD-short-kebab-title.md
```

Keep records short and practical. A useful decision explains the context, the choice, and the consequences for future work.
