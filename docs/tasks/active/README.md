# Active Tasks

This folder contains work currently in progress.

## Rules

- A task should move here from [[../backlog/README|Product Backlog]] when implementation starts.
- Keep status, implementation notes, and verification results current.
- Do not leave completed work here.
- Move verified work to [[../done/README|Done Tasks]].

## Agent Checklist

- [ ] Read the task and all linked references.
- [ ] Update the task while implementing.
- [ ] Record verification commands and results.
- [ ] Move the task to `docs/tasks/done/` when complete.
